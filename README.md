# angualr2-webpack 

```bash 
# 克隆
git clone git@github.com:tangJunGit/angualr2-webpack.git

# 进入angular2-webpack目录
cd angular2-webpack

# 添加项目所需全局依赖的库
npm install typings webpack-dev-server rimraf webpack -g

# 添加项目所需依赖的其他库
npm install

# 启动服务
npm start
```
浏览器访问地址[http://localhost:8080](http://localhost:8080)
